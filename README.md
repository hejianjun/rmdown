# REDOWN

心(jing)血(chong)来(shang)潮(nao)想要下载某网站所以的种子，于是写了个自动下载种子的工具，此工具是配合[t66y-spider](https://github.com/hejianjun/t66y-spider)使用的，目前只支持jsonlines格式

使用方法为

```bash
pip install -r requirements.txt
python rmdown.py
```

需要依赖以下模块

```py
#http请求
requests
#进度条
progressbar
#xml识别
lxml
```
